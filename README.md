## 1. 프로젝트 개요 (Overview)

만다린은 연애 관계에서 발생하는 **불확실성과 후회**로 인해 발생하는 사용자의 **감정적 비용을 최소화**하고, 최적의 행동 방향을 제시하는 **AI 기반 감정 선택 헬퍼 및 상담사** 앱 서비스입니다.

과도하게 신중하거나 충동적인 행동 패턴을 반복하여 관계의 질이 떨어지는 사용자들을 대상으로, 안전한 시뮬레이션 환경을 제공하여 건강한 관계를 구축하도록 돕습니다.


## 핵심 기능 (Core Features)

만다린이 제공하는 3가지 핵심 기능은 사용자가 후회 없는 선택을 하도록 돕는 과정입니다.

### 1️⃣ 현실의 연애 상대를 앱 내 캐릭터로 만들기 (Character Profiling)
* **설명:** 실제 연애 상대방의 성향, 말투, 대화 패턴 등 AI 모델 학습 및 프롬프트 구성을 위한 **맞춤형 프로필**을 앱 내에서 구축합니다.
* **기술적 의의:** 시뮬레이션의 현실감을 극대화하고, AI 분석의 정확도를 높이는 기반 데이터가 됩니다.

### 2️⃣ 채팅 시뮬레이션 (Chat Simulation)
* **설명:** 사용자-캐릭터 간의 대화를 진행하며, **하고 싶은 말을 하기 전**이나 **후회됐던 말을 다시 고쳐서** 해볼 수 있는 환경을 제공합니다.
* **기술적 의의:** 백엔드와 AI API의 실시간 연동을 통해 상대방의 성향이 반영된 현실적인 답변을 즉각적으로 제공합니다.

### 3️⃣ 시뮬레이션 결과 및 AI 분석 리포트 제공 (AI Analysis & Report)
* **설명:** 시뮬레이션 결과를 바탕으로 AI가 상황 분석 리포트, 연애 프로필 요약, **최적의 행동 방향에 대한 구체적인 조언** 및 시각 자료를 제공합니다.
* **기술적 의의:** 단순한 대화가 아닌, **객관적인 인사이트**를 제공하여 사용자의 의사 결정 및 선택에 대한 실질적인 힌트를 제시합니다.

## 기술 스택 (Technology Stack)

| 역할 | 기술 스택 | 세부 사항 및 선정 이유 |
| :--- | :--- | :--- |
| **Frontend** | `React Native (Expo)` | Android/iOS 크로스 플랫폼 지원을 통해 개발 효율성 극대화 및 빠른 프로토타이핑이 가능합니다. |
| **Backend** | `Spring Boot` | 확장성과 안정성이 검증된 엔터프라이즈급 백엔드 프레임워크를 경험합니다. (초보자의 학습 곡선을 감안하여 핵심 API 구현에 집중) |
| **Database** | `MySQL` | 표준 관계형 데이터베이스를 사용하여 사용자, 캐릭터, 대화 기록 등 핵심 데이터를 구조적으로 관리합니다. |
| **AI/ML** | `API Call` (LLM API) | 짧은 개발 기간 내에 고품질의 결과를 도출하기 위해, 자체 모델 학습 대신 상용 LLM API를 활용하여 **프롬프트 엔지니어링** 역량을 집중합니다. |

## 4. 서비스 아키텍처 (Architecture Overview)

만다린 서비스는 F/E, B/E, DB, AI 간의 유기적인 연동을 통해 작동하며, 특히 B/E-AI 간의 빠른 데이터 처리가 중요합니다.

본 프로젝트는 PM, F/E, B/E의 3개 주요 파트로 구성되며, B/E, DB, AI 역할은 상호 협력하는 단일 백엔드 팀으로 통합 운영됩니다.

| **F/E (Frontend Engineer)** | [이동근, 성윤수] | React Native Expo 앱 UI/UX 구현, B/E API 연동, 앱 상태 관리 |
